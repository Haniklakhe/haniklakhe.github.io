Visit the live website here: [https://haniklakhe.github.io/](https://haniklakhe.github.io/)

## ✏️ How to Edit This Website
This website comes with a built-in **Owner Mode** that allows you to edit content directly in the browser.

### 1. Enter Edit Mode
Go to your live URL and add `?owner=hanik2026` to the end:
`https://haniklakhe.github.io/?owner=hanik2026`

### 2. Make Changes
- **Text:** Click the **"✎ Toggle text editing"** button in the bottom-right panel. You can now click and type on any text on the page.
- **Photos:** Hover over any photo placeholder or existing photo and click the button to upload a new one.

### 3. Save & Publish
Since GitHub Pages is static, changes made in the browser don't automatically save to GitHub.
1. Click **"⬇ Download this page"** in the Owner Panel.
2. This will download the updated HTML file (e.g., `index.html`) to your computer.
3. Upload this new file back to this GitHub repository to make your changes live for everyone.
